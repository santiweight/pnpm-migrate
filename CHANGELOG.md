# pnpm-migrate v0.1.0

First public release.

`pnpm-migrate` migrates npm projects to pnpm from an isolated git worktree, opens a GitHub PR, and uses a coding agent to polish docs/CI and help get checks green.

```bash
curl -fsSL https://raw.githubusercontent.com/santiweight/pnpm-migrate/main/install.sh | bash
```
